1. Run 'npm install' in the project root
2. Run 'npm start' 
3. Navigate to localhost:3000 in web browser

Notes:
The following settings were changed, as follows, in Firefox by accessing 'about:config' in the address bar.
- privacy:file\_unique\_origin = false
- security.fileuri.strict\_origin\_policy = false
