# Rendering R-Markdown files to pdf (installation examples for Mac OS)

## Installation
- Make sure R is installed (else: brew install R)
- Make sure pandoc is installed (else: brew install pandoc)

## Rendering
- Assuming the R-markdown file is called `report-template.rmd`, run:

    Rscript -e "rmarkdown::render('report-template.rmd')"

- Report will be saved as `report-template.pdf`
